# Release Truthfulness Checklist

## Build and Run

- [ ] Fresh clone works.
- [ ] Dependencies install from declared files only.
- [ ] Main advertised command runs.
- [ ] README quickstart has been tested exactly as written.
- [ ] No undocumented manual steps are required.
- [ ] No hardcoded private/local paths remain.
- [ ] No missing files are referenced.
- [ ] Public demo mode does not require private credentials, tokens, or secrets.

## Documentation Honesty

- [ ] README claims match implemented functionality.
- [ ] Incomplete features are labeled incomplete, planned, optional, credentialed, or experimental.
- [ ] No roadmap item is described as current capability.
- [ ] No result/log/demo appears unless generated by this repo.
- [ ] Limitations and known failure modes are stated plainly.

## Evidence

- [ ] Test command included.
- [ ] Example command included.
- [ ] Example output can be regenerated or is marked illustrative.
- [ ] Generated artifacts are committed intentionally or excluded intentionally.
- [ ] Local/CI reproducibility note exists.
